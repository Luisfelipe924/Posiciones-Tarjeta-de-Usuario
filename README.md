# Posiciones - Tarjeta de Usuario

Este proyecto es un ejercicio práctico de **HTML** y **CSS**, enfocado en el uso de **propiedades de posición** para crear una **tarjeta de perfil de usuario** con un diseño moderno y ordenado.


## Descripción del proyecto

El objetivo de este proyecto es aplicar diferentes tipos de posicionamiento en CSS (`static`, `relative`, `absolute`, `fixed`) para organizar los elementos dentro de una tarjeta de usuario.  
Se busca lograr una composición equilibrada y atractiva utilizando solo HTML y CSS.
